Reflection:
Creating an algorithm, pseudocode, and flowchart helped me understand how to solve a problem step by step before writing a program. At first, it was a little hard to arrange the steps in the correct order, but it became easier as I practiced. Overall, I learned that planning first makes programming easier and more organized.
